# Meditation Timer App

Questo progetto è una web app sviluppata con React che permette di utilizzare un timer per sessioni di meditazione.

## Descrizione

L'applicazione è pensata per aiutare l'utente a svolgere sessioni di meditazione guidata attraverso un timer semplice e intuitivo.  
Include una pagina principale e una pagina dedicata alla meditazione.

## Funzionalità

- Timer per la meditazione
- Navigazione tra Home e pagina Meditazione
- Gestione dello stato tramite Context API
- Interfaccia semplice e minimalista

## Tecnologie utilizzate

- React
- React Router
- Context API
- CSS
